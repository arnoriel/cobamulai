# Cobamulai Landing Page

Landing page premium untuk startup jasa pembuatan AI dan website dengan tema biru modern.

## Fitur

- Desain UI/UX premium dengan tema biru (biru muda, biru, biru tua, putih)
- Responsif di semua perangkat
- Animasi halus dan interaktif
- Navigasi yang mudah
- Form kontak fungsional
- Portofolio dengan filter
- Testimoni klien
- SEO friendly

## Teknologi

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Icons
